# Contributor Code of Conduct

The R2PQ Project follows the Contributor Covenant v2.1.

We pledge to:
- Maintain a respectful, harassment-free environment
- Welcome contributions from all backgrounds
- Resolve conflicts constructively

## Unacceptable Behavior
Includes:
- Harassment or targeted insults
- Discriminatory remarks
- Personal attacks
- Spam, malware, or malicious intent

## Reporting
Send Code of Conduct concerns to:

**r2pq-security@protonmail.com**

By contributing to R2PQ, you agree to uphold this Code of Conduct.
