 Security Policy

## Reporting a Vulnerability
If you discover a security issue, please email:

**r2pq-security@protonmail.com**

Do NOT create a public issue.

We will:
- Acknowledge your report within 48 hours  
- Investigate and validate the issue  
- Work with you on coordinated disclosure  

## Supported Versions
Security patches are applied to:
- The latest stable release  
- The latest development branch  

## Scope
This policy covers:
- r2pq-sdk  
- r2pq-cli  
- r2pq-precompiles  
- r2pq-devnet  
- r2pq-bench  
- r2pq-docs  

## Appreciation
We appreciate security researchers who help keep the R2PQ ecosystem safe.
